0x01 Distance Formula From Coordinate Geomtry

  Before I give code, let me explain about formula of coordinate geometry. This formula were derived from Pythagorean to find the distance between plot and other plot graph.

  From this picture, you've seen that between 2 coordinates have hidden angle. To find distance, you have to find a side and b side to find c side (c^2 put square root) and this is proof of this formula.

a^2 + b^2 = c^2
(x2-x1) + (y2-y1) = c^2
c = sqrt((x2-x1) + (y2-y1))

** distance of a side = x2-x1
distance of b side = y2-y1
and let a,b,c to do pythagorean formula (a^2 + b^2 = c^2) and this is python code that I wrote and this is the output of distance between 2 coordinates. A(2,1), B(1,2)
